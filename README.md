# Teach Niche

Welcome to **Teach Niche**, a platform designed to empower the kendama community
by providing a space where players of all levels can share knowledge, hone their
skills, and support one another. Our mission is to foster growth, connection,
and financial sustainability within the kendama world.

## About

Teach Niche was founded by Jaymin West, a passionate kendama player with over
seven years of experience. The platform was born from a simple idea: to help
kendama players make a living from what they love. We believe in the power of
community collaboration, growth, and learning, and we are committed to providing
resources for players at every skill level.

## Features

- **Marketplace**: Browse and purchase lessons from top kendama players.
- **Lesson Creation**: Instructors can create and sell their own lessons.
- **Profile Management**: Users can manage their profiles, including updating
  personal information and viewing purchased lessons.
- **Stripe Integration**: Secure payment processing for purchasing lessons.
- **Responsive Design**: Accessible on both desktop and mobile devices.

## Getting Started

To get started with Teach Niche, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/teach-niche.git
   cd teach-niche
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**: Create a `.env` file in the root directory
   and add the necessary environment variables as shown in `.env.example`.

4. **Run the development server**:
   ```bash
   npm start
   ```

5. **Access the application**: Open your browser and navigate to
   `http://localhost:3000`.

## Project Structure

- **src**: Contains the main application code, including components, pages, and
  utilities.
- **supabase**: Contains Supabase configuration and serverless functions for
  backend operations.
- **public**: Contains static assets and the main HTML file.

## Technologies Used

- **React**: Frontend library for building user interfaces.
- **Supabase**: Backend as a service for authentication, database, and storage.
- **Stripe**: Payment processing for secure transactions.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Express**: Node.js web application framework for server-side logic.
